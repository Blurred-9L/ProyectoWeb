Cargar alumnos

Se puede cargar un archivo de alumnos. El formato del archivo es el de un archivo separado por comas (.csv), donde los valores de cada campo especifico estan separados por comas. Una vez se haya seleccionado el archivo y elegido el curso al cual se buscan agregar los alumnos, se puede hacer clic sobre el botón "Cargar" para cargar el archivo, haciendo las validaciones correspondientes. Se mostrarán la información de los alumnos que se busca añadir, donde será posible seleccionar aquellos que se quieran agregar, de ser el caso. Una vez se haya elegido, se puede presionar el botón "Registrar" para ingresar toda la información de los alumnos al sistema.
